## Installation
```
npm i mfm-js
```

## Usage
Please see [docs](./docs/index.md) for the detail.

const inputText =
`<center>

## Develop
### 1. Clone
```
git clone https://github.com/misskey-dev/mfm.js.git
```

### 2. Install packages
```
cd mfm.js
npm i
```

### 3. Build
```
npm run build
```

### Use the interactive CLI parser
full parser:
```
npm run parse
```

plain parser:
```
npm run parse-plain
```
